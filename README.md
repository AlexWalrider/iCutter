# iCutter
Программа для ресайза изображений JPEG<br>
Язык – PHP<br>
Среда разработки – PHP Devel Studio<br>

# Функционал:

- ресайз изображений<br>
- обрезка изображений<br>
- поточная обработка изображений<br>
